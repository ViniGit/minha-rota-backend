
# Back-end Minha Rota

Deve ser inserido no .env-example as credenciais que irão ser itilizadas no projeto.
    A princípio foi utilizado o docker para subir a imagem do postman.

Instalar dependências
    npm i
    yarn 

Criar tabelas com migrations 
    yarn ou npm migration:generate
    yarn ou npm migration:run

Rodar o projeto:
    yarn dev
